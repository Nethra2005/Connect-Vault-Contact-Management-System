# Connect-Vault: Contact Management System

## Overview
Connect-Vault is a web-based Contact Management System developed using Python Flask and MongoDB.

## Features
- User Registration
- User Login & Logout
- Add New Contacts
- View Contacts
- Search Contacts
- Update Contacts
- Delete Contacts
- MongoDB Integration

## Technology Stack
- Python
- Flask
- MongoDB
- HTML5
- CSS3
